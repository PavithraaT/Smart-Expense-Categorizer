# Smart Expense Categorizer 💸

AI-powered expense categorization and analytics system built using Python, Scikit-learn, and Streamlit.

---

## Features
- Categorizes transactions into Food, Travel, Shopping, Health, Education, Bills, Utilities, Entertainment
- Accepts multiple transactions at once
- Shows total spend
- Displays category-wise bar chart & pie chart

---

## Live Demo
Try the app live here:  
[Open App](https://smart-expense-categorizer-1.streamlit.app)

---

## 🖥️ Installation & Run Locally

1. **Clone the repository:**
git clone https://github.com/PavithraaT/Smart-Expense-Categorizer.git

2. **Go into the project folder:**
cd Smart-Expense-Categorizer

3. **Install dependencies:**
pip install -r requirements.txt

4. **Run the Streamlit app:**
streamlit run app.py

5. **Open the app in your browser** (usually at `http://localhost:8501/`)

## 📂 Project Files

* `app.py` → Streamlit dashboard with ML integration
* `model.pkl` → Trained ML model
* `vectorizer.pkl` → Text vectorizer for ML input
* `requirements.txt` → Python dependencies
* `screenshots/` → Folder containing app screenshots
